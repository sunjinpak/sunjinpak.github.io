# Project 5: GRIM Agriculture

**Title**: Green Initiative in Agriculture: A Qualitative Investigation of Agriculture Workers' Experiences with Air Quality in the San Joaquin Valley

**Team**: Quin

## Overview

This project analyzes interview transcripts of agriculture industry workers to understand how air quality affects their work, health, and community connections in the San Joaquin Valley. Using thematic analysis (Braun & Clarke, 2006), you will identify patterns across 8 interviews with agriculture workers and agriculture-adjacent employees.

## 2. Thematic Analysis Guide

### What is Thematic Analysis?

Thematic analysis is a method for finding **patterns (themes)** in interview data. You read through transcripts carefully, label important parts with "codes," and then group those codes into bigger themes.

Think of it like sorting laundry. Each piece of clothing is a quote from an interview. You sort them into piles (codes), and then you group similar piles together into categories (themes).

### Step 1. Read All Transcripts (Familiarization)

**Goal**: Get to know the data before you start analyzing.

1. Read each transcript from start to finish without highlighting anything
2. On a second read, start noting things that stand out
3. Write down your initial reactions

**Tips**:
- Read at least twice before coding
- Ask yourself: "What is this person telling me about their experience?"
- Pay attention to strong emotions, surprising statements, and repeated ideas

### Step 2. Generate Initial Codes (Open Coding)

**Goal**: Label meaningful segments of text with short descriptive tags.

1. Go through each transcript line by line
2. When you find something interesting, highlight it and write a short label (code)
3. One quote can have multiple codes

**Example**:

| Quote | Code |
|-------|------|
| "I see there like a lot of pesticides that get purchased" | pesticide exposure |
| "When they're harvesting the little particles of pistachios go into the air. So I wear masks" | harvest dust, protective behavior |
| "Our spray rig operators, we'll get them tested to see if they're allergic" | employer health screening |
| "I currently work as a field supervisor for an agricultural company" | outdoor work identity |

**Tips**:
- Be specific. "Bad air" is too vague. "Dust from almond harvest" is better
- Use the participant's own words when possible (called "in vivo" codes)
- Keep a running list of all your codes in a spreadsheet

### Step 3. Search for Themes (Grouping Codes)

**Goal**: Sort your codes into groups that represent bigger ideas.

1. Print or list all your codes
2. Look for codes that are related to each other
3. Group related codes and give each group a name (this becomes a theme)

**Example**:

```
Theme: "Pesticide Awareness and Exposure"
  - pesticide exposure
  - crop chemical knowledge
  - seasonal spraying
  - protective gear use

Theme: "Employer Response to Air Quality"
  - employer health screening
  - mask provision
  - schedule flexibility
  - lack of employer action
```

**Tips**:
- Aim for 4-7 themes
- Some codes might not fit anywhere. That's OK
- Themes should be distinct from each other

### Step 4. Review Themes

**Goal**: Check if your themes make sense with the data.

**Level 1**: Re-read quotes under each theme. Do they belong?
**Level 2**: Re-read entire transcripts with themes in mind. Did you miss anything?

### Step 5. Define and Name Themes

For each theme, write:
1. **Theme name** (short, descriptive)
2. **Definition** (1-2 sentences)
3. **Best quotes** (2-3 representative quotes)

For each theme:
1. Introduce the theme
2. Present 2-3 supporting quotes
3. Explain what the quotes show
4. Connect to the next theme
